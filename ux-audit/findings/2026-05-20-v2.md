## 남은 개선 제안 (우선순위)

### 🟡 Medium
1. **`/log` 목록 썸네일 부재** — 홈 일지 피드는 사진을 보여주는데 `/log` 목록은 여전히 텍스트만(`v2-log-list.png`). 사진 중심 서비스 정체성과 불일치. → 목록 카드에 썸네일 추가. 파일: `app/log/Log.tsx`, `components/share/LogCard.tsx`(summary 분기)
2. **목록 전체 일괄 렌더** — `/log` 목록이 항목 전체를 한 번에 렌더(무한스크롤/페이지네이션 부재). 데이터 증가 시 비용↑. → 목록에도 무한스크롤 적용. 파일: `app/log/Log.tsx`, `app/recipe/Recipe.tsx`

### 🟢 Low
3. **검색·필터 부재** — 레시피/일지 목록 탐색 수단 없음(Stage2 '검색'). 데이터 증가 대비.
4. **LCP 이미지 priority** — 피드 첫 이미지 priority 미지정 경고. `components/share/RecipeCard.tsx`
5. **(dev 전용) Amplitude 키 누락 콘솔 에러** — 운영 무관, dev env 설정 시 노이즈 제거.

> 참고: 1차에서 'off-brand'로 적었던 우하단 열대 섬 아이콘은 **TanStack Query 개발자도구 버튼**(dev 전용)으로 확인됨 → 운영 UI 아님, finding 무효.
